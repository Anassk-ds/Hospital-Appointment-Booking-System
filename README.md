Hospital Appointment Booking System

Overview

The Hospital Appointment Booking System is a web-based application developed using HTML and CSS. It allows patients to view doctor information and book appointments online through a user-friendly interface.

Features

- Hospital introduction and facilities section
- Doctors information table
- Online appointment booking form
- Emergency contact information
- Hospital introduction video
- Emergency audio announcement
- Google Maps location integration
- Form validation using HTML5
- Responsive and clean user interface using CSS

Technologies Used

- HTML5
- CSS3

Modules

About Hospital

Provides information about the hospital and available facilities.

Doctors Information

Displays doctor details including:

- Doctor Name
- Specialization
- Experience
- Available Days

Appointment Booking Form

Allows patients to enter:

- Name
- Email
- Mobile Number
- Age
- Gender
- Doctor Selection
- Appointment Date
- Symptoms

Emergency Contact

Displays:

- Ambulance Number: 108
- Emergency Helpline: 112

Multimedia

- Hospital Introduction Video
- Emergency Audio Announcement

Location

Google Maps integration for hospital location.

Form Validation

The application uses HTML5 validation:

- Required Fields
- Email Validation
- Mobile Number Pattern Validation
- Age Range Validation
